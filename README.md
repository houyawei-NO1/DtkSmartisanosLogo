DtkSmartisanosLogo

目标依旧是为了学习DTK，获取锤科SmartisanOS系统绘制Logo

目前只有一个新建文件夹

![新建文件夹](https://github.com/houyawei-NO1/DtkSmartisanosLogo/blob/main/%E8%AE%BE%E8%AE%A1%E5%9B%BE.jpg)

v0.1

单应用图标下载

1：选择下载模式

2：输入应用包名

3：选择下载位置

4：预览并下载重绘图标
